指令

指令的四种调用方法

        E: 元素调用
        <my-directive></my-directive>
        A: 属性调用
        <div class="my-directive"></div>
        C: 类调用
        <div class="my-directive"></div>
        M: 注释调用
        <!--directive:my-directive-->

绑定策略

        <div my-directive
            my-url="http://www.baidu.com"
            mylinktext="Click me to go to baidu"
        </div>

        angular.module('myApp', [])
        .directive('myDirective', function() {
            return {
                restrict: 'A',
                replace: true,
                scope: {
                    myUrl: '@',
                    myLinkText: '@'
                },
                template: '<a href="{ {myUrl} }">' + '{ {myLinkText} }</a>'
                }; 
        });

常用实用ng指令(在之前项目较少用到,在此整理一下)

        ng-cloak

        除了使用ng-bind来避免未渲染元素闪烁,还可以在含有{{ }}的元素上使用ng-cloak指令:
        示例:
        <p ng-cloak>{ { greeting } }</p>
            <body ng-init="greeting='HelloWorld'">
        </body>
        ----------------------
        ng-bind-template

        同ng-bind指令类似，ng-bind-template用来在视图中绑定多个表达式。
        示例:
        <div ng-bind-template="{ {message} }{ {name} }">
        </div>
        ----------------------
        ng-form

        ng-form用来在一个表单内部嵌套另一个表单.普通的HTML <form>标签不允许嵌套，但ng-form可以.
