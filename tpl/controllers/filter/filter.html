过滤器


所有字母大写

{{uppercaseText | uppercase}}

在JS中过滤所有字母小写

{{lowercaseText}}

限制小数点后的位数

{{ 123.456789 | number:2 }}

将一个数值格式化为货币格式

{{ 123 | currency }}

将日期格式化成需要的格式

{{ clock.now | date:'medium' }}
{{ clock.now | date:'short' }}
{{ clock.now | date:'fullDate' }}
{{ clock.now | date:'longDate' }}
{{ clock.now | date:'mediumDate' }}
{{ clock.now | date:'shortDate' }}
{{ clock.now | date:'mediumTime' }}
{{ clock.now | date:'shortTime' }}

对年份进行格式化

{{ clock.now | date:'yyyy' }}
{{ clock.now | date:'yy' }}
{{ clock.now | date:'y' }}

对月份进行格式化

{{ clock.now | date:'MMMM' }}
{{ clock.now | date:'MMM' }}
{{ clock.now | date:'MM' }}
{{ clock.now | date:'M' }}

对日期进行格式化

{{ clock.now |date:'dd' }}
{{ clock.now | date:'d' }}
{{ clock.now | date:'EEEE' }}
{{ clock.now | date:'EEE' }}

对小时进行格式化

{{clock.now | date:'HH'}}
{{clock.now | date:'H'}}
{{clock.now | date:'hh'}}
{{clock.now | date:'h'}}

对分钟进行格式化

{{ clock.now | date:'mm' }}
{{ clock.now | date:'m' }}

对秒进行格式化

{{ clock.now | date:'ss' }}
{{ clock.now | date:'s' }}
{{ clock.now | date:'.sss' }}

对字符进行格式化

{{ clock.now | date:'a' }}
四位时区标识: {{ clock.now | date:'Z' }}

函数过滤

过滤出数组中含有e的元素
{{ ['Ari','Lerner','Likes','To','Eat','Pizza'] | filter:'e' }}
根据字典中的某一个字段进行过滤
{{ [{ 'name': 'Ari', 'City': 'San Francisco', 'favorite food': 'Pizza' },{ 'name': 'Nate', 'City': 'San Francisco', 'favorite food': 'indian food' }] | filter:{'favorite food': 'Pizza'} }}
在JS中通过方法进行过滤
{{ ['Ari','likes','to','travel'] | filter:isCapitalized }}

json

json过滤器可以将一个JSON JavaScript对象转换成字符串,便于调试
{{ {'name': 'Ari', 'City': 'SanFrancisco'} | json }}

limitTo

截取字符串的前三个字
{{ "San Francisco is very cloudy" | limitTo:3 }}
截取字符串的后六个字
{{ "San Francisco is very cloudy" | limitTo:-6 }}
截取数组中的第一个元素
{{ ['a','b','c','d','e','f'] | limitTo:1 }}

number

千位符号,如果传入了一个非数字字符,会返回空字符串
{{ 123456789 | number }}

orderBy

对象数组用name字段进行排序
{{ [{ 'name': 'Ari', 'status': 'awake' },{ 'name': 'Q', 'status': 'sleeping' },{ 'name': 'Nate', 'status': 'awake' }] | orderBy:'name' }}
将第二个参数设置为true可以将排序结果进行反转
{{ [{ 'name': 'Ari', 'status': 'awake' },{ 'name': 'Q', 'status': 'sleeping' },{ 'name': 'Nate', 'status': 'awake' }] | orderBy:'name' : true }}

自定义过滤器

将一句话的首字母转换成大写
{{ "ginger loves dog treats" | lowercase | capitalize }}
